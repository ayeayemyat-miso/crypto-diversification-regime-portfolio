# Data Information

The data used in this project includes:

- Bitcoin (BTC-USD) daily prices
- Ethereum (ETH-USD) daily prices
- S&P 500 (^GSPC) daily index values
- US 3-Month T-Bill yield (risk-free rate)

In this public repository, data is obtained programmatically from free sources
such as Yahoo Finance and FRED using Python. Licensed Refinitiv DataStream
data used in the original MSc thesis is **not** redistributed.
